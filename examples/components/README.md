# BlenderUI Components example

This standalone application is the searchable, interactive component catalog.
It presents live controls, editor primitives, dialogs, property templates, and
their usage guidance without the full Blender workspace shell.

Run it from this directory:

```sh
flutter pub get
flutter run -d macos
```

The same target can be launched from the repository root with:

```sh
flutter run -d macos -t examples/components/lib/main.dart
```
