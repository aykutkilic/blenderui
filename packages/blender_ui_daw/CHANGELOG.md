# Changelog

## 0.1.0

- Introduced DAW project/session models and editing controllers.
- Added arrangement, MIDI, waveform, automation, mixer, transport, plug-in
  browser, and plug-in rack widgets.
- Added the native plug-in host contract and deterministic in-memory host.
- Added native CoreAudio and CoreMIDI adapters plus Audio Unit capability and
  VST3 discovery metadata support.
- Added retained editor-type selectors to every dock area.
- Added track/time scaling, non-destructive trim/stretch/offset/tempo controls,
  loop regions, expanded automation editing, and scale-aware piano editing.
- Added audio-device preferences plus synchronized track/master effect-chain
  and node-graph editors with portable master-chain persistence.
- Added typed browser-to-chain drag insertion, compact per-device bypass/VU
  chrome, transactional host/project updates, and six built-in processors.
